# Origin Client — Minecraft 1.21 build

The Fabric build of Origin Client for Minecraft 1.21 (Java 21). Byte-identical
source to the 1.21.1 module — 1.21 and 1.21.1 share the same pre-1.21.2
GuiGraphics.blit API, so there are no render-layer changes; only the build
coordinates and the standalone install model differ (perf/shader stack comes
from the launcher's catalog, not jar-in-jar).

## Setup

For setup instructions, please see the [Fabric Documentation page](https://docs.fabricmc.net/develop/getting-started/creating-a-project#setting-up) related to the IDE that you are using.
